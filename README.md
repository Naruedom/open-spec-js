# Open Spec Js README

## Extension for open file spec for .js, .ts
## How to use
1. open vscode navigation

    WINDOWS
    ```
    CTRL + SHIFT + P
    ```
    
    Mac
    ```
    CMD + SHIFT + P
    ```

2. type : "OpenSpecJS: Toggle"


----

## Use shortcut

WINDOWS
```
SHIFT + ALT + S
```

Mac
```
CTRL + SHIFT + S
```
